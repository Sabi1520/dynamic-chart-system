## Dynamic Chart System

## Overview
A dynamic chart generation and visualization system developed using Python. The project generates interactive charts and visual representations based on input datasets.

## Features
- Dynamic chart generation
- Data visualization
- Interactive graph rendering
- Multiple chart support
- Simple and user-friendly interface

## Technologies Used
- Python
- Matplotlib / Plotly / Pandas (whichever you used)

## How to Run

1. Install dependencies
pip install -r requirements.txt

2. Run the application
python app.py
